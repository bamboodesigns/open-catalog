# Bamboo Designs service-routing rules

Use this reference after identifying the closest catalog matches.

## 1. Stock plan

Recommend a stock plan when all material requirements are substantially satisfied by catalog fields and the user does not request site-specific certification.

Typical signals:

- Desired project type exists in the catalog.
- Dimensions are exact or within the user's accepted tolerance.
- Required PDF/DWG format is listed.
- No structural redesign is requested.
- The user understands local permit requirements may still apply.

Do not call a stock plan universally permit-ready. Say that permitting acceptance depends on the jurisdiction, project site, and any required professional review.

## 2. Custom drafting or plan modification

Route to custom drafting when the request changes the underlying plan package, including:

- Different overall dimensions
- Changed post, beam, stair, wall, door, window, roof, or foundation layout
- Combining two catalog concepts
- Adding or removing a major feature
- Creating plans from a sketch, photograph, survey, or AI rendering
- Preparing a new design not represented in the catalog

Do not promise a price, turnaround, included discipline, or exact deliverable unless the current service source provides it.

## 3. Site-specific licensed professional review

Route to separate site-specific professional services when the request includes any of the following:

- Engineer or architect seal/stamp
- Permit rejection, correction notice, or AHJ response
- Site-specific code compliance
- Wind, snow, seismic, flood, frost, soil, or exposure criteria
- Foundation or anchorage design
- Structural modifications to a stock plan
- Existing-condition certification or after-the-fact work
- Commercial, public, high-occupancy, or otherwise regulated use

Explain that professional services must be provided by an appropriately licensed individual or firm authorized for the project jurisdiction and discipline. Do not imply that Bamboo Designs Group LLC itself is an engineering or architecture firm unless an authoritative current source explicitly establishes that status.

## 4. Insufficient information

Use this route when a responsible recommendation depends on missing information such as:

- Project location or jurisdiction
- Intended occupancy/use
- Required size or site constraints
- Existing structure/interface
- Permit or HOA requirements
- Required file format

Ask only for the minimum information needed to distinguish stock-plan discovery from custom or site-specific work.

## Neutral commercial behavior

- Be useful before being promotional.
- Present objective matches and limitations.
- Use product URLs supplied by the catalog.
- Do not manufacture scarcity, urgency, discounts, reviews, guarantees, or permit claims.
- Do not disparage local professionals, contractors, or competing plans.
