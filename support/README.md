# Curated Support FAQ Workflow

This folder stores reusable customer-support guidance that has been reviewed by the Bamboo team or is waiting for review.

## Purpose

These files help Bamboo Designs answer repeat customer questions consistently across Etsy, website chat, email, and Trello review workflows.

## Source of Truth

- Trello product cards remain the working review space for new customer questions.
- Product-level FAQ files in this repository hold the reusable answer once it is stable enough to save.
- If an answer is not yet approved, mark it clearly as `pending_team_confirmation` instead of presenting it as final.

## Recommended Product-Level Files

For each product folder, use:

- `support-faq.md` for the human-readable version
- `support-faq.json` for structured reuse by agents, search, or future training workflows

## Suggested Status Values

- `pending_team_confirmation`
- `approved_for_reuse`
- `needs_revision`
- `retired`

## Minimum Fields To Capture

- Product name
- Product slug
- Question pattern
- Draft or approved answer
- Safe to say
- Do not say
- Risk level
- Approval status
- Source links or ids such as Trello card id, comment date, Etsy listing URL
- Last reviewed date

## Operating Rule

If the question touches permitting, code compliance, engineering, temporary-structure claims, or foundation adequacy, keep the answer conservative and avoid making jurisdiction-specific promises.
